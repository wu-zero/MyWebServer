# 压力测试
## 测试环境
* OS：Ubuntu16.04
* 内存：252G
* CPU：	Version: Intel(R) Xeon(R) CPU E5-2678 v3 @ 2.50GHz

## 测试方法
* 选择本地环境测试
* 线程池开启4线程，总共用5个cpu核心
* 压测软件`webbench`
## 线程查看
```bash
top -H -p <pid>
```
